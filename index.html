A description about myself.

Using OOP To Create Highly Organized Code

This blog post covers how I used Object Oriented Programming in my Full-Stack JavaScript Single Page Application with a Rails backend. I chose to build a complex card game in which players compete to place all their sets first. Using Classes I was able to create highly organized code, making a complex idea into a comparatively easy build.

My Approach to Learning New Languages/Frameworks

In this blog post I discuss my plans to learn the MERN stack now that I have become a solid Full-Stack Ruby Developer. I talk about how I am approaching Node.js and the MERN stack. Approaching additional languages/frameworks is far easier now that I know the concepts of Frontend/Backend development solidly in one language.

Using Scope Methods to Narrow Database Queries

In this blog post I discuss how I defined scope methods in my Ruby model files and used them in order to make more technically difficult database queries which led to an improved user experience. The post goes into detail about the differences between Rails and SQL queries and shows how I used table joins within my scope methods to arrange courses by number of reviews.

Pushing Yourself in Full-Stack Development

This blog post discusses my first Full-Stack application which was coded using Sinatra. It goes into detail about how I pushed myself much further than the project requirements and the process of researching and using additional gems and frameworks in order to create efficient and clean code that implemented advanced features into my application.

Third Party APIs and Metaprogramming

In this blog post I discuss how to work with Third Party APIs and my use of the fantasy NBA API in my CLI Project. I also discuss Metaprogramming. Code that writes code is an extremely efficient way of programming and I use it in conjunction with my API data in order to seemlessly create my database of NBA players and create an application which solves a real-world issue.

"Why did you decide to learn Software Engineering?"

In my first blog post I discuss how I took a career break and after trying out multiple career based courses on Udemy, ultimately decided to pursue a career in Software Engineering. I discuss the similarities between linguistics (my major in college) and coding and talk about what the future holds for me now that I have been accepted into the Flatiron Bootcamp.
